# Mobile Preview Instructions

To view the app in phone-sized preview:

## Chrome/Edge:
1. Open the app at http://localhost:8081
2. Press `Cmd+Option+J` (or `F12`) to open DevTools
3. Press `Cmd+Shift+M` to toggle device toolbar
4. Select a device preset (iPhone 12 Pro, iPhone SE, etc.) from the dropdown

## Safari:
1. Open the app at http://localhost:8081
2. Press `Cmd+Option+I` to open Web Inspector
3. Click the device icon in the toolbar to enable responsive design mode
4. Select a device preset

## Firefox:
1. Open the app at http://localhost:8081
2. Press `Cmd+Option+I` to open Developer Tools
3. Click the responsive design mode icon (or press `Cmd+Shift+M`)
4. Select a device preset

## Quick Mobile Viewport Sizes:
- iPhone 12/13/14 Pro: 390 x 844
- iPhone SE: 375 x 667
- Samsung Galaxy S20: 360 x 800
- Pixel 5: 393 x 851

You can also manually set the viewport size in the device toolbar.
